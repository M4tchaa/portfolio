# Overview

Tick is a countdown timer application designed specifically for event organizers and timekeepers. It provides customizable timer scenes, automated scene transitions, and cue notes to help presenters and event crews manage sessions more efficiently during live events.

# Architecture

* Next.js Web UI for the user interface
* Multi-scene timer management system
* Automated scene switching and cue note support
