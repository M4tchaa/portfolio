# Overview

ERP Prioritas Group is an internal enterprise resource planning system developed to streamline and centralize company operations across multiple business processes. The system is designed to improve operational efficiency, data management, and workflow coordination within the organization.

# Architecture

* CodeIgniter 3 as the backend framework
* PostgreSQL for database management
* jQuery and AJAX for dynamic client-side interactions
* Modular web-based ERP architecture for internal operational workflows
