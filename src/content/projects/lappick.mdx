# Overview

LapPick adalah sistem rekomendasi laptop berbasis NLP yang menggunakan pendekatan TF-IDF untuk mencocokkan kebutuhan pengguna dengan spesifikasi laptop yang tersedia.

# Architecture

- TF-IDF untuk pemrosesan teks
- Flask API sebagai backend inference
- Next.js Web UI untuk antarmuka pengguna
